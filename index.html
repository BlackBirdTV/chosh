What is CHOSH?
CHOSH is a basic, bash-like shell written in C#. It uses the chronoTerminal text-based interface, the sublib parser and QUSO for extensions. CHOSH supports inline C#, Python and Powershell code and so called functions (commands that can be used in every script and in chronoTerminal) written in C#, Python or CHOSH. CHOSH is (mostly) open-source, issue reports can and feature requests can be put onto the Git Repo.
